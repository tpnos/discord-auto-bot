# Project Rules



- This is a Discord bot project.
- 
- Use Node.js and discord.js.
- 
- Keep all secrets in .env or Railway Variables.
- 
- Never expose Discord bot token.
- 
- Before changing code, explain the plan briefly.
- 
- After changing code, run tests or at least `npm run start` check.
- 
- Preserve existing features unless explicitly asked.
- 
- Make small, safe commits.
- 
- Follow the structure requested by the user for autonomous self-correction.
